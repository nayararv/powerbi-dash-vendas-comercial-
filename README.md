# 📊 Dashboard de Vendas

Este dashboard está sendo desenvolvido em **Power BI** com o objetivo de analisar dados de vendas e gerar insights que auxiliem na tomada de decisão.

## 🚧 Status do Projeto
O dashboard encontra-se **em desenvolvimento**. Novas visualizações, métricas e melhorias de design serão adicionadas conforme o projeto evolui.

## 🗄 Fonte dos Dados
Os dados utilizados neste dashboard foram criados e modelados em **SQL**.

A base de dados utilizada neste projeto está disponível no repositório:

➡️ https://github.com/nayararv/sql-analise-de-vendas

Neste repositório foram desenvolvidos:

- Modelagem do banco de dados
- Criação das tabelas
- Relacionamentos entre entidades
- Inserção de dados simulados de vendas
- Consultas SQL para análise de dados

## 📈 Objetivo do Dashboard

Apresentar análises visuais sobre:

- Total de vendas
- Evolução das vendas ao longo do tempo
- Produtos mais vendidos
- Desempenho por vendedor
- Tendências de vendas mensais

## 🛠 Ferramentas Utilizadas

- **SQL (MySQL)** — criação e modelagem da base de dados  
- **Power BI** — construção do dashboard e visualizações  
- **GitHub** — versionamento e portfólio

## 📌 Observação

Os dados utilizados neste projeto são **dados simulados**, criados com fins educacionais e para desenvolvimento de **portfólio profissional em análise de dados**.
